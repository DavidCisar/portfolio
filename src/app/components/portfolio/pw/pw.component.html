3D Portfolio Website

Description


Thank you for visiting my portfolio website! I worked on this project entirely in my free time, trying to combine my interest in 3D modeling and frontend development within a professional portfolio website.

As a “Quereinsteiger” without an academic background in computer science, it is utterly important to prove your expertise through showcasing selected realized projects. To satisfy this, I built this creative portfolio website, which not only displays information about me, my skillset or recently build projects.

It explicitly allows guests like you to experience an immersive interaction within a more or less cozy 3D model, rather than just receiving information.

The portfolio website is a pure frontend project build with Angular. With the help of the open-source 3D modeling software Blender and the selected corresponding web framework Three.js, I built an immersive 3D portfolio website that invites guests to interact with the model by exploring this virtual three-dimensional room.

As you might have noticed, I'm an enthusiastic web3 and blockchain developer in my free time. Since these technologies like tokenization are used more and more professionally, I intend to develop myself further and acquire new tools and areas of application.

Technologies

Angular, TypeScript/JavaScript, Blender, Three.js

Timeframe

August - December, 2022
